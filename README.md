# MakeUp - Ultra Fast
Ultra fast Minecraft shader (Java). For a small graphic improvement. Intended for very low-spec computers.

## Main features
* Antialiasing (FXAA)
* Depth of field
* Water reflection and refraction
* Auto-exposure
* Shader for all dimensions
* FAST!

Tested on Nvidia and Intel, under Linux.
